## Usage

```jsx
// Regular button
<Button
  buttonVariant="primary"
  icon={<SearchIcon height={20} width={20} />}
  size="middle"
  onClick={onClick}
>
  Search
</Button>

// Icon button
<Button
  buttonVariant="icon"
  icon={<SearchIcon height={20} width={20} />}
  size="middle"
  onClick={onClick}
/>
```
